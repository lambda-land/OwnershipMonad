A Haskell DSL Demonstrating a Rust-lang inspired variant of Affine Logic

Author: Mike McGirr
ONID: mcgirrm
Oregon State University


This project uses the Stack tool (which is similar to Cabal)
It will handle all the necessary external libraries.

To build run the command
$ stack build

To run the main file run the command
$ stack exec OwnershipMonad-exe

The project code is implemented in the src/LibDataMap.hs file
The example programs are located in the app/Main.hs file
